# Mooney – Smart Budget Tracker 🦆💰

Mooney is a web-based expense tracking application designed to help users manage their spending, stay within their monthly budget, and develop better financial habits.

The app allows users to track expenses across categories, monitor overspending, and receive insights into where their money is going.

---

## Features

- Add and track expenses easily
- Create custom spending categories
- Set monthly budget limits
- Identify overspending patterns
- Simple and interactive user interface
- Smart budgeting insights

---

## Tech Stack

- Next.js
- React
- TypeScript
- CSS
